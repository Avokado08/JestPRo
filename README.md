# Metalworker Web Application

Современное веб-приложение (Landing Page) для мастера-жестянщика.

## Инструкции по запуску для Пользователя (Если устанавливается Node.js)
1. Установите [Node.js](https://nodejs.org/) (рекомендуется версия LTS).
2. Запустите консоль в папке проекта.
3. Выполните команду `npm install` для установки зависимостей.
4. Выполните команду `npm run dev` для запуска локального сервера разработки.

## Инструкции для Агентов (Instructions for Agents)
- **Стек:** React 18, Vite, Vanilla CSS.
- **Архитектура:** Все компоненты находятся в `src/components`. Глобальные стили в `src/styles`.
- **Дизайн:** Строгий индустриальный. Темный фон, металлические текстуры, оранжевые/желтые акценты.
- При разработке новых компонентов обязательно используйте CSS-переменные из `src/styles/variables.css`.
- Следуйте плану разработки в `task.md`. Никаких лишних библиотек CSS (Tailwind, Bootstrap) — стилизация выполняется чистым CSS для максимального контроля над 3D-анимациями и тенями.
